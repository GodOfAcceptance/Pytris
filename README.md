A Tetris environment that implements Gymnasium's Env class. This project allows you to play Tetris on your own, but does not have any AI agents.
Tetris features includes 7-bag, SRS, DAR, ARR. 
Action space is a set of binary vectors of dimension 7, representing key presses each frame. 
Observation space is a tuple of board, next-pieces, hold-piece, currently falling piece.
